2.Maximum Number of Words
A sentence is a list of words that are separated by a single space with no leading or trailing spaces.
You are given an array of strings sentences, where each sentences[i] represents a single sentence.
Return the maximum number of words that appear in a single sentence.
Example 1:
Input: sentences = ["alice and bob love leetcode", "i think so too", "this is great thanks very much"]
Output: 6
Explanation:

- The first sentence, "alice and bob love leetcode", has 5 words in total.
- The second sentence, "i think so too", has 4 words in total.
- The third sentence, "this is great thanks very much", has 6 words in total.
  Thus, the maximum number of words in a single sentence comes from the third sentence, which has 6 words.

6.Character Frequency
Problem Statement:
Given a string, create an array of size 26, where each element represents the frequency of a character from 'a' to 'z' in the given string.
Constraints:
1 <= n <= 10^5
The input string consists only of lowercase English letters.
Input Format:
First line containing n length of string
Second line containing a string of lowercase English letters.
Output Format:
An array of size 26, where the i-th element represents the frequency of the (i+1)-th character in the English alphabet.
